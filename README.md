# Chef Hussaini's E-Commerce UI 🍔

> A high-fidelity, responsive restaurant e-commerce web interface.

**🔴 [View Live Demo](https://haipedia.github.io/food-app)**

## About The Project
This project is a front-end simulation of a modern restaurant ordering portal. It was engineered to showcase responsive UI/UX principles, modern DOM structuring, and interactive e-commerce mechanics.

## Tech Stack
* **HTML5:** Semantic markup and structure.
* **CSS3:** Custom styling, Flexbox/Grid layouts, and responsive media queries.
* **JavaScript:** Interactive elements and simulated ordering calculators.

## Key Features
* Fully responsive layout optimized for both mobile and desktop viewing.
* Simulated shopping cart and checkout form UI.
* Dynamic ingredient calculator interaction.
